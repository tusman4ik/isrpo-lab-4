## План тестирования

### 1. Цели и задачи тестирования

**Основная цель:** Проверить корректность работы математических функций библиотеки geometric_lib для расчета площади и периметра геометрических фигур.

**Задачи тестирования:**
- Проверить корректность расчета площади круга и квадрата, обработку граничных случаев  
- Обеспечить автоматизацию процесса тестирования
- Документировать результаты тестирования

### 2. Описание тестируемого продукта

Продукт: Библиотека geometric_lib - математическая библиотека для работы с геометрическими фигурами.

**Функциональность:**
- Расчет площади круга по радиусу
- Расчет периметра (длины окружности) круга по радиусу
- Расчет площади квадрата по длине стороны
- Расчет периметра квадрата по длине стороны

**Требования к продукту:**
- Корректность математических расчетов согласно геометрическим формулам
- Обработка некорректных входных данных (отрицательные значения)

### 3. Область тестирования

**Модули, подлежащие тестированию:**
- `geometric_lib/circle.py`
  - Функция `area(r)` - расчет площади круга
  - Функция `perimeter(r)` - расчет периметра круга
- `geometric_lib/square.py`
  - Функция `area(a)` - расчет площади квадрата  
  - Функция `perimeter(a)` - расчет периметра квадрата

### 4. Стратегия тестирования
**Типы тестирования:**
- Unit-тестирование отдельных функций
- Функциональное тестирование математических расчетов
- Тестирование граничных значений

**Методы:** Автоматическое тестирование с использованием unittest

### 5. Критерии приемки
- 100% успешное выполнение всех unit-тестов
- Обработка всех исключительных ситуаций
- Соответствие результатов математическим формулам

### 6. Ожидаемые результаты
- Отчет о прохождении тестов
- Исходный код тестов в директории `tests/`
- Документация по использованию библиотеки

## Реализация тестов

Исходный код тестов размещен в репозитории GitHub. В истории коммитов отражено добавление тестовых файлов:

- `tests/test_circle.py` - тесты для функций работы с кругом
- `tests/test_square.py` - тесты для функций работы с квадратом

### Команды для запуска тестов:

```bash
python -m unittest discover -s tests -v
```
